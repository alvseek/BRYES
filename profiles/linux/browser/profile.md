# Web Browser (Google Chrome, Linux)

Profile for browsing the web in Google Chrome on the Linux desktop container.
Inherits `profiles/linux/profile.md`. Site-specific profiles (e.g.
`profiles/linux/browser/tokopedia/profile.md`) inherit this.

## Terms & Vocab
- Address bar (omnibox) — the wide bar at the TOP of the Chrome window showing the current URL; typing a URL + Enter navigates there.
- Tab strip — the row of page tabs just above the address bar.
- Nav buttons — Back (left arrow), Forward, Reload (circular arrow), to the LEFT of the address bar.
- Page content — the scrollable web page BELOW the address bar; this is the website itself.
- On-page search box — a search field provided by the WEBSITE (distinct from the browser address bar) for searching within that site.

## Visual
- You are looking at Google Chrome. The address bar at the very top is the BROWSER; everything below it is the WEBSITE.
- To read the current site, look at the Page content, not the address bar.

## Operating
- To open a website: click the Address bar, type the full URL (e.g. "tokopedia.com"), then press Enter.
- To search WITHIN a site, use the site's own on-page Search box — NOT the address bar.
- Pages often lazy-load: after navigating or searching, wait a moment and scroll down to reveal more content before deciding what is there.
- Prefer the site's on-page controls over the browser chrome for any in-site task.
